# Statcheck

To run statcheck, you either need to [install statcheck in R](https://mbnuijten.com/statcheck/) and use R:

```{r}
statcheck::statcheck("F(..., ...) = ..., p < 0.05", OneTailedTests = FALSE,pEqualAlphaSig = FALSE,OneTailedTxt = FALSE)
```
or (recommended) use the Statcheck: The Game web app at [https://sachaepskamp.shinyapps.io/statcheck/](sachaepskamp.shinyapps.io/statcheck/) (also usuable on mobile devices)!
